事前準備として、GitHub のアカウントを作成します。

GitHub はソフトウェア開発のプラットフォームで、ソースコードを保存することができます。

<br>

### # アカウント登録



以下のページからアカウント登録が可能です。

https://github.com/join




**Email preferences** はチェックを入れなくても大丈夫です。

アカウントを作成したら、登録したメールアドレスに確認メールが届きます。

メール認証が完了したら、準備はOKです。
